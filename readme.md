# Описание решения задачи.
1. Генерируется массив случайных чисел заданной длины.
2. Создаётся буфферный массив той же длины, переменная счётчика чётных чисел в исходном массиве, изначально равная 0.
3. Проходится цикл, в ходе которого каждое число массива по очереди проверяется на чётность. Если число чётное - оно записывается в буфферный массив, а к счётчику прибавляется 1.
4. Если счётчик чётных чисел не равен нулю, создаётся финальный массив с длинной, равной значению счётчика. Если счётчик равен нулю, то создаётся пустой массив.
5. В созданный массив заносятся все чётные числа из буфферного массива.
6. Возвращается финальный массив.
